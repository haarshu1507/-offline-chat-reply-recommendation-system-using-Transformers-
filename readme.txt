Chat Reply Recommendation System
--------------------------------

How to Run:
1. Open ChatRec_Model.ipynb in Jupyter Notebook.
2. Ensure datasets are in this path:
   /Desktop/Dataset/userA_chats.csv
   /Desktop/Dataset/userB_chats.csv
3. Run all cells step-by-step.
4. The model will generate a reply for test input.
5. The trained model is saved as Model.joblib.

Libraries Required:
- pandas
- torch
- transformers
- joblib

Output:
Predicted reply text printed in console.
